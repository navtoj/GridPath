# Grid Path

### Max-Value Path Problem

- Start at top left. `(0, 0)`
- Finish at bottom right. `(n-1, n-1)`
- Each coordinate has a value: `positive`, `negative`, or `zero`.
- Do not revisit a coordinate.
- Do not move diagonally.
